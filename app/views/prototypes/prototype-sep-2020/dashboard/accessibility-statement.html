Accessibility statement for the Online Civil Money Claims service

This service allows you to make a money claim against an individual or an organisation for £10,000 or less.

This website is run by HM Courts and Tribunals Services (HMCTS). We want as many people as possible to be able to use it, so we’ve designed it to be accessible. That means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the website text as simple as possible to understand.

AbilityNet (opens in a new window) has advice on making your device easier to use if you have a disability.

How accessible is this website?

The website has been tested by the Digital Accessibility Centre (DAC) and it meets the required level of accessibility.

Some PDFs, documents and email attachments may not be fully accessible. For example, the forms and guidance documents which can be downloaded. These will be tested and made accessible by September 2020.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS) (opens in a new window).

If the claim goes to a hearing

We’ll tell you the hearing centre you need to go to and give you information about its accessibility.

Technical information about this website’s accessibility

HMCTS is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

The website is fully compliant with the Web Content Accessibility Guidelines version 2.0 AA standard (opens in a new window).

We are in the process of testing the full service against the new Web Content Accessibility Guidelines version 2.0 AA standard (opens in a new window). We commit to being fully compliant against this standard by 2020.

Non-accessible content

Some PDFs, documents, and attachments to email notifications may not meet accessibility standards or work with a screen reader.

This is because they were created before current accessibility legislation was introduced. We plan to make them accessible by September 2020.

How we tested this website

We’ve carried out manual accessibility testing using multiple in-browser accessibility tools. Axe and Wave (tools) comply with WCAG 2 and Section 508 accessibility (which covers WCAG 2.1)

In addition to this, the Online Civil Money Claims website was tested by the Digital Accessibility Centre (DAC) in February 2018.

We continually run automated accessibility testing as part of our development practices. This helps us to resolve issues before we launch new pages or features. We use Voice Over and Axe as part of our quality assurance processes.

This is a summary of the DAC Report from Feb 2018. Many of the high to medium issues mentioned have now been resolved.

“The Digital Accessibility Centre (DAC) user/technical team carried out an accessibility audit for the Ministry of Justice’s Civil Money Claims service on 13th February 2018. This document incorporates the findings regarding any accessibility barriers identified during the process.

There were areas of the service that caused some difficulties for our analysts. Screen reader software reads the content on a page in a linear fashion and this also applies to layout tables, unless table headers are used. Table headers ensure that content is read out in columns and rows. Tables were encountered that contained headers that were layout tables and vice versa, which cause confusion for our screen reader analyst.

All of the error skip links functioned as expected; however, the third-party area that users were taken to did not possess one. There is no ‘skip to content’ link is present on the ‘Court and tribunal form finder’ page. A skip to content link would usually benefit users by navigating their focus to the primary content of the page.

Links were present that were not descriptive to users that rely on audio feedback, whether navigating within the context of the page, or out of context (viewing by links alone). More description/context is required to ensure that users that rely on audio feedback can navigate with confidence. Form fields had not been labelled in some instances and others had labels hidden off screen, which were read to screen reader users, but were of no benefit to cognitive or dyslexic users.

As a result of updating page content, user focus was taken to the top of the page, resulting in keyboard users needing to re-navigate the page in order to return to the point at which they lost focus; this can be extremely frustrating to keyboard only users and disorienting to screen reader and screen magnification user groups.“

What we’re doing to improve accessibility

This website is continually tested using accessibility software. Any new features which are introduced are also tested.

We will be making the PDFs, documents and attachments that are essential to using our service accessible.

We aim to carry out an accessibility audit with DAC in the near future.

This statement was prepared and published on 23 September 2019.
